# Dochacat:3 CatTower

Welcome to the official documentation for **Dochacat:3 CatTower**!

---

## Installation

The first step is to download Dochacat:3 CatTower. You can get it [here](https://github.com/Villylily/Dochacat-CatTower/releases). *(Don't forget to add your actual download link here!)*

Once you have your file, drag it to your preferred location and extract it. Open the extracted folder, double-click the `StartDochacat.bat` file, and run it.

---

## Configuration

**Important:** Close the application, then navigate to the `data` folder and open the `config` file. It should look like this:

`{
  "port": 3000,
  "registrationOpen": true,
  "invitePassword": "CHANGE_ME",
  "globalMute": false,
  "maintenanceMode": false,
  "giphyApiKey": ""
}`

Here, you can change your port, change the invite password, and add a Giphy API key if you have one. DO NOT touch the rest of the file. You can easily edit those remaining settings (and the invite key) directly from the admin panel later.

Now, double-click the StartDochacat.bat file to run it again. You now have your Dochacat:3 CatTower server up and running!
# Admin Panel

To access the admin panel, simply Ctrl + Click the link in the CMD window that says "Admin Panel". It's probably pretty empty right now.

Things you can do here include triggering a lockdown (Maintenance Mode), editing users, and more.

    ⚠ Security Warning
    You can add your friends as admins, BUT THIS CAN DECREASE YOUR SECURITY. If your friend's password gets leaked, someone could mess with your server. However, they cannot remove or add other admins, and they can never lock you out of the admin panel.

 Navigation & Wrap-up

The Home Page is the next place you'll see after the login screen. It has a few apps for helpful stuff, the admin app, chat/calls, and more.
